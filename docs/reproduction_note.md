# Reproduction Note

The repository is organized according to the thesis workflow.

1. Data preprocessing and batch correction
2. Feature gene selection
3. Masked MLP model training
4. SHAP interpretation and multi-model voting
5. Orthology and synteny analysis
6. Ka/Ks selection pressure analysis
7. Promoter cis-element analysis
8. Motif and phylogenetic analysis
9. qRT-PCR visualization

Some large raw genome or sequencing files are not included in this repository. They can be downloaded from public databases according to the descriptions in the thesis.
